# Pokédex Pro

Neon-themed Pokémon Pokédex built with Vite, React, TypeScript, TailwindCSS, Zustand and Framer Motion.

## Run

```bash
npm install
npm run dev
```

## Included

- Infinite scroll Pokédex
- Search and filters by type, generation and ability
- Pokémon details modal with stats, abilities, evolution tree, moves and type effectiveness
- Favorites persisted in localStorage
- Team builder with coverage summary
- Battle Simulator Lite with turn-based combat
🚀 Pokedex-Pro

Uma Pokédex moderna, responsiva e orientada à experiência do usuário, desenvolvida com foco em performance, escalabilidade e design refinado.

O projeto consome dados em tempo real da API pública de Pokémon, apresentando informações completas de forma estruturada e visualmente agradável. A interface foi pensada para ser limpa, intuitiva e eficiente, permitindo navegação fluida entre gerações, busca dinâmica e visualização detalhada de cada Pokémon.

Cada entidade possui tipagem destacada, descrição contextualizada, estatísticas representadas visualmente e uma organização que prioriza legibilidade e hierarquia de informação — princípios essenciais em UX/UI de alto nível.

A arquitetura do código segue boas práticas de desenvolvimento, incluindo:

Separação de responsabilidades
Uso de cache para otimização de requisições
Renderização progressiva para melhor performance
Manipulação eficiente do DOM

Este projeto demonstra não apenas domínio técnico em HTML, CSS e JavaScript, mas também atenção a detalhes de produto, experiência do usuário e qualidade visual — aproximando-se de aplicações reais utilizadas em ambiente profissional.
